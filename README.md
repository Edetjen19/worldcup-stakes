# World Cup 2026 — what's at stake

A simple, mobile-first viewer for the 2026 World Cup group stage: the games, what each one means in plain English, and who's going through, still alive, or knocked out — including the race for the 8 best third-place "wildcard" spots.

Single static page, no build step, no dependencies. Open `index.html` or visit the hosted version.

Data is a snapshot captured 25 June 2026 from public sources (Wikipedia, ESPN, FIFA, Sky, FOX). Not affiliated with FIFA.
